Getting Started

This section will guide you to set up Ganeti - please check back for future updates!

For the following distributions you may use your package manager to install the Ganeti packages:

  • Debian
  • Ubuntu
  • Gentoo
For Redhat/CentOS based systems, please head over to this repository.

We also provide Ubuntu packages through this PPA.
Additionally, we provide nightly packages for Debian and Ubuntu off the project's master branch via this Repository. These nightly packages are only for testing purposes, do not use them in production! There is no upgrade path between nightly builds!
